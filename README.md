# 🫀 Heart Disease Analytics Dashboard

A web-based analytics platform for exploring **heart disease patterns** using interactive data visualization.

This project integrates **Python Flask**, **Tableau**, and modern **web technologies** to present healthcare data in a structured and visually intuitive dashboard.  
The system allows users to explore relationships between different health indicators and cardiovascular risk factors.

## 📌 Project Overview

Heart disease is one of the leading causes of mortality worldwide. Healthcare datasets contain multiple indicators such as **age, BMI, physical activity levels, and lifestyle habits** that influence cardiovascular health.

The objective of this project is to analyze these indicators using **interactive visualizations**, allowing users to explore patterns and correlations that may contribute to heart disease risk.

The system integrates a **Tableau visualization dashboard** within a **Flask-based web interface**, creating a simple and accessible platform for healthcare data analysis.

---

## ✨ Key Features

- Interactive **Heart Disease Analytics Dashboard**
- Visualization of **health indicators and risk factors**
- **KPI summary cards** providing quick dataset insights

- **Dark mode support** for improved usability

## ⚙️ Installation & Setup

Follow these steps to run the project locally.

### 1. Clone the Repository

```bash
git clone <repository-link>
cd heart_disease_project
```

### 2. Install Dependencies

```bash
pip install flask
```

### 3. Run the Application

```bash
python app.py
```

### 4. Open the Application

Open your browser and navigate to:

```
http://127.0.0.1:5000
```
