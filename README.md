# Heart Disease Analytics Dashboard

A web-based analytics platform for exploring patterns in heart disease data using interactive visualizations.

This project combines **Python Flask**, **Tableau**, and **web technologies** to present healthcare data in a structured and visually intuitive way. The dashboard allows users to analyze relationships between different health indicators and heart disease risk factors.

## Project Overview

Heart disease is one of the leading causes of mortality worldwide. Healthcare datasets contain multiple factors such as age, BMI, physical activity levels, and lifestyle habits that influence cardiovascular health.

The objective of this project is to analyze these indicators through **interactive data visualizations**, allowing users to explore patterns and relationships that may contribute to heart disease risk.

The system integrates a **Tableau visualization dashboard** within a **Flask-based web interface**, providing a clean and accessible platform for data analysis.

---

## Features

- Interactive **Heart Disease Visualization Dashboard**
- Analysis of **health indicators such as BMI, age, and lifestyle factors**
- **Key statistics overview** using KPI cards
- Embedded **Tableau analytics dashboard**
- Responsive **web interface with sidebar navigation**
- **Dark mode support**
